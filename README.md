# Crypto Flow Radar 加密資金流雷達

## 專案定位

Crypto Flow Radar 是一個加密貨幣市場環境判讀工具。

它不是用來直接告訴使用者「買」或「賣」，而是用來整理：

- 美國宏觀環境
- 加密市場資金流
- 合約 Funding / OI
- 市場情緒
- 板塊輪動
- 過熱風險
- 策略環境

核心理念：

> 不是告訴你能不能進場，而是告訴你現在適合用什麼方式進場。

## 主要功能

### 1. 今日重點摘要

快速顯示：

- 今日市場狀態
- 市場總分
- 主要推動來源
- 行情健康程度
- 主要風險等級
- 策略環境
- 風險提醒

### 2. 宏觀判讀

整合：

- CPI
- PPI
- PCE
- NFP
- 失業率
- DXY 美元指數
- 10Y 美債殖利率
- Nasdaq 100 / QQQ

其中 CPI / PPI / PCE / NFP / 失業率目前透過 `macro_input.json` 手動維護。  
DXY / 10Y / QQQ 透過 yfinance 自動抓取。

### 3. 加密資金流

整合：

- BTC 價格
- ETH 價格
- BTC Funding
- ETH Funding
- BTC OI
- ETH OI
- 24H 成交額
- 恐懼貪婪指數

### 4. 資金流來源模型

用來判斷行情主要偏向：

- 現貨買盤推動
- 合約槓桿推動
- 市場情緒推動
- 板塊輪動推動
- 資金動能不足

### 5. 板塊交叉驗證雷達

本工具不直接用自定義板塊取代第三方分類。

目前採用：

- CoinGecko 第三方分類資料
- Binance 代表幣 24H 交易資料

進行交叉驗證，並顯示：

- CoinGecko 分類狀態
- 代表幣追蹤狀態
- 代表幣數
- 一致性
- 可信度
- 綜合狀態

### 6. 圖表總覽

包含：

- 資金流來源模型分數圖
- BTC / ETH Funding 對比圖
- 強勢板塊 Top 8 漲跌圖

### 7. 資料來源狀態
